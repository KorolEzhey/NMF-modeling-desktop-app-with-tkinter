# РГР: Выделение независимых признаков (NMF)

Расчётно-графическая работа по теме 7, глава 10 книги Тоби Сегарана «Программируем коллективный разум».

## Описание

Desktop-приложение на Python с графическим интерфейсом (tkinter), реализующее **ручную версию алгоритма неотрицательной матричной факторизации (NMF)** для автоматического выделения скрытых тем из набора текстовых документов.

## Архитектура проекта

| Файл | Назначение |
|---|---|
| `main.py` | Точка входа |
| `nmf_core.py` | Реализация NMF с нуля: `difcost()`, `factorize()` — итеративная минимизация ошибки реконструкции |
| `preprocessing.py` | Токенизация, лемматизация (pymorphy3), удаление стоп-слов, построение матрицы «документы × слова» |
| `data.py` | Корпус из 70 русских новостных сниппетов (спорт, политика, экономика, наука) |
| `gui.py` | Интерфейс на tkinter: параметры NMF, список текстов, вкладки с темами, документами, графиком и матрицей |

## Запуск

```bash
# Установка зависимостей
pip install numpy pymorphy3 matplotlib

# Запуск приложения
python main.py
```

## Как работает

1. **Предобработка**: тексты разбиваются на слова, лемматизируются, фильтруются стоп-слова и редкие/частые термины.
2. **Матрица V**: формируется матрица частот слов по документам.
3. **NMF**: матрица V факторизуется на произведение W × H, где W — веса тем в документах, H — ключевые слова тем.
4. **Интерпретация**: для каждой темы выводятся топ-слова из H и топ-документы из W.
5. **Сохранение**: результаты экспортируются в `features.txt` и `articles.txt`.

## Скриншоты / результат

Пример выделенных тем (pc=4, iter=100):

- **Тема 1** — Спорт (общий): победа, одержать, благодаря, этап, мировой...
- **Тема 2** — Спорт (соревнования): метр, дистанция, сто, финал, чемпионат...
- **Тема 3** — Политика (выборы и законы): партия, объявить, избирательный, закон...
- **Тема 4** — Политика (правительство): провести, правительство, требование, бюджет...
